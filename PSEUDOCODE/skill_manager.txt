
Define function to add skills to selected character:
    - Use menu function from main to choose between adding an existing skill, new one, or return

    - If return chosen:
        - return to start of skill management function

    - If existing skill option is chosen:
        - access dictionary with saved skill named 'saved_skills'
        - print all saved skills from saved_skills dictionary
        - use menu function form main to ask which skill to add or return
        - if return chosen then return to start of skill managing function
        - if skill selected then add skill to character skills in the characters main attribute list
        - return to start of skill managing function

    - If new skill option is chosen:
        - let user input name(Save skill name temporarily) with option to return(go back to start of skill managment function)
        - let user input description(Save skill description temporarily) with option to return(go back to start of skill managment function)
        - let user input effect(Save skill effect temporarily) with option to return(go back to start of skill managment function)
        - let user input target(Save skill target temporarily) with option to return(go back to start of skill managment function)
        - save skill to dictionary saved_skills and apply to character
        - return to start of skill management function

Define function to remove skills from a character:
    - Use menu function from main to choose between all characters skills and return(which returns to skill managment function)
