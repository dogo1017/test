def add_menu function with parameter characters
- set new_character to a dictionary with all their changeable stuff
- set new_character's name to user input
- set new_character's class to user input
- set new_character's level to user input's integer
- set new_character's race to user input
- always loop
    - ask user for an attribute or type "esc" to go on.
    - if user input is "esc"
        - break out of loop
    - else add attribute to new_character's attributes
- always loop
    - ask user for an skill or type "esc" to go on.
    - if user input is "esc"
        - break out of loop
    - else add skill to new_character's skills
- always loop
    - ask user for an inventory item or type "esc" to go on.
    - if user input is "esc"
        - break out of loop
    - else add inventory item to new_character's inventory
- append new_character into characters
- return characters
